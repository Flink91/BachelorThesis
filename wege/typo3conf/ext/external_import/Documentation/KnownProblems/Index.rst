.. _problems:

Known problems
--------------

Namespace support for XML data type was introduced for a project
within a very precise scope. It was tested only within that scope and
should be considered as being in "beta" state at best. Namespace
support is not trivial and can certainly be improved.

Please report bugs and improvements as usual to:
`http://forge.typo3.org/projects/extension-external\_import/issues
<http://forge.typo3.org/projects/extension-external_import/issues>`_.
